About

My final project for CSCI 202, Dynamic Web Pages. The Persona Compendium is a collection of all the Personas from Persona 3 Reload, using a Json file by Luyluish (Credits at bottom). You can sort through the collection by specifying what reactions that the persona has to certain types of damage.

Instructions

Clicking on the question mark images will change the type of reaction you are sorting through for that individual element. For example, if you want to find personas which are weak to fire, click the symbol until it is an exclamation mark, for weak. Then search to see the list of personas! Furthermore, clicking on a particular persona that is displayed will open a new tab with more in-depth information about that persona. Pressing reset will reset the reactions back to default (Unknown) and remove the current personas shown on screen. Here is a Key for the reactions and elements:

Unknown: Reaction not yet found

Weak: Takes increased damage

Nullifies: Takes no damage

Reflects: Turns damage back on attacker

Absorbs: takes the incoming damage and converts to health for itself

Resists: Takes less damage
